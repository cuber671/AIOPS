# AIOps 项目与论文集合

本仓库整合了AIOps领域的相关论文与多个实践项目，涵盖根因分析、智能监控、因果发现等核心方向，为AIOps研究与开发提供参考。

## 项目结构

```
.
├── paper/       # AIOps相关论文（含算法类、LLM类、综述类）
└── projects/    # AIOps实践项目（含实验平台、智能体、评估工具等）
```

## 说明

- 论文按研究方向与年份分类整理，项目包含多种AIOps核心场景实现。
- 各子目录下均有专属说明，具体安装与使用请参考对应目录文档。
